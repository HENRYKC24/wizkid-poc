## Getting Started
### Get A Local Copy And Run Application Locally
```bash
git clone https://github.com/CyberFuzzion/Wizkid-POC-Frontend.git

cd Wizkid-POC-Frontend

npm install

npm start

scan the QR with your android or iOS phone

```

## Test Application

```bash
npm test
```
